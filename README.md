# Queima Turbo Max 2.0

Site oficial do programa **Queima Turbo Max 2.0**, desenvolvido por Fabiola Feliciano. Focado em transformação corporal para mulheres na menopausa através de um método prático, rápido e eficiente.

## 🚀 Tecnologias Utilizadas

- **React 19**
- **Vite** (Build ultrarrápido)
- **Tailwind CSS 4** (Estilização moderna e leve)
- **Framer Motion** (Animações fluidas)
- **Lucide React** (Ícones leves)

## ⚡ Performance e Otimização

O projeto foi construído com foco total em performance:
- **Lazy Loading**: Imagens carregam apenas quando necessário.
- **Video Preload**: Vídeos carregam metadados primeiro para não travar o início da página.
- **Animações Otimizadas**: Uso de GPU via Framer Motion para transições suaves.
- **Responsividade Total**: Ajustado para Mobile, Tablet e Desktop.

## 🛠️ Como rodar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Rode em modo de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Gere o build para produção:
   ```bash
   npm run build
   ```
   Os arquivos otimizados serão gerados na pasta `/dist`.

## 📂 Estrutura de Arquivos

- `src/`: Código fonte do projeto.
- `public/`: Mídias (Fotos e Vídeos) otimizados.
- `src/App.tsx`: Estrutura principal da Landing Page.
- `src/index.css`: Configurações globais e animações customizadas.
